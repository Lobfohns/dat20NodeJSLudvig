Terminal Commands I can look up quickly whenever I need them

How to add your work to already existing github repo in terminal


    git commands. (steps to push to a repository)
    step 1. git add -A (adds all the files, can use "." then it only adds the files where you are located)
    step 2. git status (not neccessary to be done)
    step 3. git commit -m "yourmessage"
    step 4. git push
    git pull to pull the repository

How to install express when you open up a project for the first time, you need express in dependencies


    {
        "dependencies": {
            "express": "this should suggest whatever is latest version, use that." 
        }
    }
    then you can do the following in terminal:
    npm install express

How to run your node project with example filename "app.js" and a few navigation terminal commands


    node app.js to run node project
    nodemon app.js to run node project with nodemon
    npm run "script-name" to run script
    Make sure you are in the right directory use "cd" to change directory
    Use "ls" to get a list of all files in current folder
    Use "cd .." to go up one directory

How to create svelte project


   npm degit sveltejs/template "name of your project"
   cd to "name of your project"
   npm install
   npm run dev

   Write "--single inside start in package.json to specifiy single page app"
