A little bit about different theory subjects we have learned about in class

Server Side Rendering

Server Side Rendering is used in order for a user to not get sent a html file, it is already made in the backend. One way to do this is by concatinating in a JavaScript file. See codesnippets for an example.

Import/Export with Node Modules

You can use node modules to import and export, can be used to create routers as an example.

REST API CRUD

CRUD = Create, Read, Update, Delete.

REST API Conventions Order of methods - Get, Post, Put, Patch, Delete.

Endpoint should match a specific standard, so you understand what it does just by looking at it collection, table should always be plural in the database and REST API.

Why don't we like for loops in Javascript

Overly verbose, Error Prone, Not functional.

Code conventions / Clean code

Make sure code is similar in the whole project. For example if you have a space before your curly brackets in a function make sure it is the same way in all your functions. Spaces in the project should also be similar.
